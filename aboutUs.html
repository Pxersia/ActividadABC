Sobre Nosotros

Somos un equipo que desarrolla el proyecto ABC Challenge, combinando creatividad y tecnología para crear un abecedario visual.

Nuestro objetivo es representar cada letra del abecedario mediante imágenes de distintos elementos como objetos, animales, lugares y referencias digitales, fortaleciendo el trabajo en equipo y el aprendizaje de la programación web.

A través de este proyecto practicamos HTML, CSS y JavaScript, además de herramientas colaborativas como Git y GitHub.
