#  ActividadABC

Proyecto web hecho con HTML, CSS y JavaScript, enfocado en la práctica de estructura de páginas

---

##  Descripción

Este proyecto es una actividad de aprendizaje donde se creó una página web con distintas secciones, estilos personalizados y funciones simples con JavaScript.

---

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

##  Estructura del proyecto

- `index.html` → Página principal  
- `aboutUs.html` → Sección sobre nosotros  
- `contactUs.html` → Formulario de contacto  
- `indexStyle.css` → Estilos principales  
- `aboutStyle.css` → Estilos sección About  
- `contactStyle.css` → Estilos contacto  
- `script.js` → Lógica en JavaScript  

---

##  Funcionalidades

- Diseño de múltiples páginas  
- Estilos personalizados por sección  
- Interacción básica con JavaScript  
- Navegación entre páginas
